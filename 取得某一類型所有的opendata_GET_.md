###### tags: `opendata`


# 取得某一類型所有的opendata"GET"

> http://127.0.0.1:5000/opendatas/3
> 取得闢謠(3)的所有opendata

## 回傳
| 欄位   | 中文 | 資料型態 |
| ------ | ---- | -------- |
| status | 狀態 | bool     |
| msg    | 訊息 | str      |
| data   | 資料 | object   |


## data內容備註
* 編號1：藥品
* 編號2：食藥新聞
* 編號3：闢謠
* 編號4：藥局

## 範例(以下為闢謠，其他自行查看)

    "status": true,
